## Available Scripts

In the project directory, you can run:

### `npm start`

In the api folder, you can run:

### `json-server-auth  --watch db.json -p 5000 -r routes.json`

You can reset port:

Find:

### `sudo lsof -i :5000`

Kill:

### `kill -9 <PID>`
